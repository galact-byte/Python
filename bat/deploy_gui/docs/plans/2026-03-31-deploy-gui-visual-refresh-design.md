# Deploy GUI Visual Refresh Design

**Date:** 2026-03-31
**Scope:** `deploy_gui` 桌面应用 GUI 美化

## 目标

在不改变部署逻辑的前提下，把当前界面从“功能拼装感”提升为“现代卡片式桌面应用”。默认浅色，日志区保留深色终端风格。

## 设计方向

- 主体采用浅灰背景与白色卡片分区
- 左侧项目列表做成资源导航栏
- 中间配置区做成多卡片工作台
- 右侧计划预览与日志做成独立阅读面板
- 保留现有三栏结构，但增强标题、分组、间距、圆角和按钮层级

## 视觉原则

- 默认浅色，不做全局暗色主题
- 卡片圆角、边框、轻阴影建立层次
- 主按钮高亮，辅助按钮中性
- 日志区单独深色，形成“终端”语义
- 弹窗、输入框、下拉框、复选框统一风格

## 不在本次范围

- 不重写部署流程
- 不增加复杂动画
- 不新增主题切换功能
- 不改数据模型和 SSH 逻辑
