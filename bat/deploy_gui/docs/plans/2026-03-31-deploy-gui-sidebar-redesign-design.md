# Deploy GUI Sidebar Redesign

**Date:** 2026-03-31
**Scope:** `deploy_gui` 左侧导航栏重设计

## 目标

把当前左栏从“卡片堆叠的操作区 + 被挤压的项目列表”改成更像现代桌面应用的一体化侧栏。重点解决操作区和列表互相抢空间的问题，不改中间与右侧的工作区逻辑。

## 设计方向

- 页面类型：工具型桌面应用
- 语言：简体中文
- 风格：柔和分段式侧栏
- 原则：信息结构优先，减少套娃卡片，强化项目导航主视图

## 结构方案

- 左栏是一整块浅色导航面板，而不是多个白卡拼接
- 顶部只保留一块紧凑工具头
- `Program 预设 / 新增 / 复制 / 删除` 改成桌面工具风格的小按钮组
- 项目列表占据左栏主空间，成为真正导航区
- 列表项做成清晰的导航卡片，突出选中态与模式标签

## 避免项

- 不再把操作区做成多层卡片堆叠
- 不再添加重复摘要块
- 不再让说明文案挤占左栏高度
- 不把中间和右侧一并重做，避免扩大范围
