# 🎬 视频质量检测工具

> 扫描本地视频文件，检测真实分辨率、帧率、码率等信息，支持 Video2X AI 超分辨率 + 帧插值修复。

## ✨ 功能特性

### 📊 视频质量检测
- **批量扫描** — 选择文件夹自动扫描所有视频（支持递归子目录）
- **详细信息** — 分辨率、帧率、编码格式、码率、时长、文件大小
- **智能分级** — 自动分类 4K / 2K / 1080p / 720p / 480p，彩色标签一目了然
- **假分辨率检测** — 码率过低自动标黄 ⚠ 警告（如低码率"假4K"）
- **统计概览** — 实时统计各分辨率等级的视频数量

### 🤖 AI 视频修复（Video2X）
- **超分辨率** — 支持 RealESRGAN（真人/动漫）、RealCUGAN（动漫锐利）
- **帧插值** — 支持 RIFE 多版本（v4.26 / v4.6 / v4.25-lite）
- **智能跳过** — 已达标视频（4K / 60fps）自动跳过
- **编码可选** — H.264（兼容好）/ H.265（体积小）
- **GPU 加速** — 支持选择 GPU 设备

### 🎨 界面美化
- 4 款毛玻璃主题（深色 / 浅色 / 暗夜紫 / 翡翠绿）
- 自定义背景图片 + 可调透明度
- 右键菜单（打开文件夹 / 复制路径 / 单独修复）

## 📦 依赖

| 包名 | 用途 |
|------|------|
| `PyQt6` | GUI 界面 |
| `opencv-python` | 视频信息读取 |
| `Video2X`（可选） | AI 超分 + 插帧修复 |

## 🚀 使用方法

**方式一：双击启动脚本**
```
视频质量检测.bat
```
脚本会自动检查 Python 环境和依赖，缺少时自动安装。

**方式二：直接运行**
```bash
python video_inspector.py
```

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `video_inspector.py` | 主程序（PyQt6 GUI） |
| `视频质量检测.bat` | Windows 一键启动脚本 |
