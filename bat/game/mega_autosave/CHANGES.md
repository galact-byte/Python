# 修改记录 — Mega 自动保存助手

> **修订记录**
>
> - v1.1: 增加点击节奏控制（弹窗渲染等待 + 点击后冷却），避免点太快
> - v1.0: 初版，监控「另存为」弹窗自动点击保存

## 新增文件

### mega_autosave.py — 核心脚本

- **功能**：后台监控 Windows 原生「另存为」对话框，弹出后自动点击 [保存(S)]。
- **实现原理**：
  - `EnumWindows` 扫描可见窗口，按窗口类 `#32770` + 标题完全等于「另存为」定位弹窗（不误匹配「确认另存为」）。
  - `EnumChildWindows` 在弹窗内找文本以「保存」开头的 Button。
  - 用 `SendMessage(BM_CLICK)` 跨进程点击，无需把窗口切到前台。
- **节奏控制**：`SETTLE_DELAY` 弹窗出现后先等渲染稳定再点；`COOLDOWN` 每次点完冷却；都在文件顶部可调。

### start.bat — 启动器

- 仅做引导：设代码页 + 标题，调用 `mega_autosave.py`。

## 测试方式

1. 双击 `start.bat` 启动，看到「监控中...」。
2. 在 Mega 触发一次下载，等「另存为」弹窗出现，确认约 0.6 秒后被自动点击保存。
3. 控制台出现 `[1] 已保存 时间` 即正常；按 Ctrl+C 停止。
