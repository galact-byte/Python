## v2.1 — 精准填充修复

### 核心修复：doc_writer.py 重写

- **勾选框**：通过修改 Wingdings sym 元素（`0030`→`F0FE`）实现真实打勾，不再替换文本
- **地址填空**：只替换模版中的空格占位 run（省、市、县、详细地址各自独立），保留"省(自治区、直辖市)"等模版文字
- **表标题**：`表二（ / ）` → `表二（定级对象名）`，不再用项目名
- **封面**：自动填写"备案单位"
- **附件名**：勾选"有/无"的同时在"附件名称"后精准填入名称，不替换整行
- **上级主管**：无时默认填"/"
- **数据总量/月增长量**：只替换下划线占位处

### 核心修复：doc_reader.py 增强

- **勾选项读取**：新增 `_read_checked_options()` 函数，扫描 sym 元素读取已勾选项
- **兼容两种勾选码**：同时匹配 `F0FE` 和 `00FE`
- **地址精准解析**：`_parse_address_runs()` 从 run 结构提取省/市/县/详细地址
- **调查表读取**：`read_survey_docx()` 从基本情况调查表提取拓扑图图片和描述文字

### 前端修复

- **地址拆分**：省、市、县、详细地址四个独立输入框
- **调查表字段**：Step 1 新增调查表文件路径（自动从父目录扫描）
- **勾选自动选中**：隶属关系、单位类型、行业类别自动匹配下拉值
- **Flask 红字**：抑制 Werkzeug WARNING 输出

## 测试方式

1. 双击 `start.bat`，浏览器自动打开
2. 粘贴旧项目目录路径，点"扫描"
3. 确认备案表、定级报告、调查表三个文件自动检测到
4. 点"下一步"，确认数据自动填充（蓝色高亮）
5. Step 3 点"预览备案表"用 Word 打开验证：
   - 封面有单位名称
   - 表标题有定级对象名
   - 地址在正确位置，模版文字不变
   - 隶属关系/单位类型等正确打勾 ☑
   - 定级报告有/无正确勾选
   - 附件名称在正确位置
6. 点"预览定级报告"验证拓扑图插入
7. 确认后点"正式生成"
